14.7.2.2 APP_TEST_Mode_T

C

typedef enum APP_TEST_MODE_T
{
    PER_TEST_MODE,
    PERIOD_TEST_MODE,
    CONT_TEST_MODE,
    PROM_TEST_MODE,
    THROUGHPUT_TEST_MODE,
}APP_TEST_MODE_T;

Description

The definition of different application Test modes. Please refer Operating Modes of application and Command Table section for more details.

Enumerator Description
PER_TEST_MODE PER TEST MODE
PERIOD_TEST_MODE PERIOD TEST MODE
CONT_TEST_MODE CONT TEST MODE
PROM_TEST_MODE PROM TEST MODE
THROUGHPUT_TEST_MODE THROUGHPUT TEST MODE
