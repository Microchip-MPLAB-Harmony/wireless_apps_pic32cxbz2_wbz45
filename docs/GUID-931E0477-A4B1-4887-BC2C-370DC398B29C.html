Programming using IPECMD.exe

Image(*.hex) file can be programmed into the device

Hardware Requirements

  • One WBZ451 Curiosity Board

  • USB Cables

  • PC

Software Requirements

  • MPLABX IDE/IPE

  • MicrochipUtilityTool.exe

Procedure

  • Follow the steps shown in the following picture

Figure 1. Figure 13: IPECMD.

  • Click on Browse File Button and select an image file.

  • Selected file will appear in the File Text Box.

  • Click on Flash Button

  • Wait for Programming to complete

    • Status: Done

Summary

  • MPLABX IPE should be installed on the PC
