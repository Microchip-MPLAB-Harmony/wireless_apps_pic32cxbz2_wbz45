5.9.2 FFD Project Verification

Procedure For Provisioning

Connect the FFD device to the USB port. We can know the status of the device via the debug prints produced on the tera term tool. From the below image:

  1. BLE advertising has started.
  2. FFD waits for provisioning parameters.

Open the mobile app and select BLE Provisioner.

The device name will be shown on the mobile app. Select the device based on device name.

Change the parameters (PAN ID and Channel) as per requirement.

Select “COMMISSION” option on the mobile app.

Now the device was commissioned successfully.

Procedure For Factory Reset

In factory reset process we are going to erase the device provisioning data and reset the device.

After clicking the factory reset, “System reset” will be invoked, so the device will disconnect from the BLE. After the successful completion of the factory reset, the device status will be switched to a not commissioned state.

Procedure For RGB LED Control

We have the capability to manage the RGB LED on the Coordinator through the MBD Mobile Application.

Open the mobile app and select BLE Sensor.

Then select PIC32CXBZ

Click on Scan option available in top right corner of the screen

Select the MicrochipFFD device

Activate the RGB LED by turning on the LED Status feature within the interface.

Please select your preferred color from the LED Color section of the interface, and the RGB LED on the Coordinator will adjust accordingly.
