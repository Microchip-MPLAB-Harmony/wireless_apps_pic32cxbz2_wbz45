Getting Started With Multiprotocol Applications

Documentation describing steps to test and develop the precompiled BLE and Zigbee coexistence application examples.

The Multiprotocol applications provide to demonstrate the capability of our PIC32CXBZ2 device. The device supports both BLE and ZigBee technologies in the 2.4 GHz frequency range. Our software stacks handle both technologies as specified in the respective specifications. Mainly, our software stack is capable to handle both technologies together in a coexistence mode. This is recommended for users who are new to the Microchip BLE stack and other toolchains like MPLAB Code Configurator.
