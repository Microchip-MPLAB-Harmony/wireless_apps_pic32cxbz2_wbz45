Hardware

The WBZ451 Curiosity Board is an efficient and modular development platform that supports rapid prototyping and demonstrates the features, capabilities and interfaces of Microchip's BLE+Zigbee RF Module WBZ451. This board offers integrated programming/debugging features using PICkit™ On-Board (PKOB4) debugger, and requires only a micro-USB cable to power-up and program the board. Users can expand its functionality through MikroElectronika mikroBUS™ Click™ adapter boards and do rapid prototyping utilizing the BLE+Zigbee enabled RF Module.

The WBZ451 Curiosity Board supports a variety of applications such as wireless lighting, home automation/Internet of Things (IoT), industrial automation and other BLE or Zigbee related applications.

Features

  • WBZ451 BLE+Zigbee RF Module

  • USB or Battery Powered

  • On-board Programmer/Debug Circuit using PKoB4 based on Microchip SAME70 MCU

  • Microchip MCP73871 Li-Ion/LiPo Battery Charger with Power Path Management

  • On-board USB to UART Serial Converter with HW Flow Control based on Microchip MCP2200 mikroBUS™ Socket to Expand Functionality using MikroElectronika Click™ Adapter Boards RGB Lighting LED connected to PWM

  • Reset Switch

  • One User Configurable Switch

  • One User LED • 32.768 kHz Crystal

  • Microchip SST26VF064B, 64 Mbit External QSPI Flash

  • Microchip MCP9700A, Low Power Analog Voltage Temperature Sensor

  • 10 pin ARM SWD Header for External Programmer/Debugger
