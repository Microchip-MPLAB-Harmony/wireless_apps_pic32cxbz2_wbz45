Getting Started with IEEE 802.15.4 P2P Phy Application

This document provides setup details and configuration for P2P PHY Application.This is an application built on top of Standalone IEEE 802.15.4 PHY layer. It enables two PIC32CXBZ2 devices to communicate and exchange data wirelessly. The application receives the data from UART and sends them over the air and similarly send the data to UART which is received by the transceiver.
